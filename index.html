Version 1.0.0

ABOUT

入力内容を外部送信しない、広告なしのQRコード生成ツールです。入力した内容は端末の外に出ません。
— 詳細 —

DevToolsのNetworkタブを開いて通信が発生しないことをご確認いただけます。

PRIVACY

QRに埋め込む入力内容（テキスト・パスワード・連絡先など）は、サーバーへの送信はもちろん、localStorage 等への保存も一切行いません。設定（テーマ・サイズ等）のみ端末内に保存されます。

USAGE

TYPEを切り替えて、テキスト / URL / Wi-Fi / メール / 電話 / SMS / vCard / 位置情報 のQRを生成できます。生成されたQRはSVG / PNGで保存、またはクリップボードにコピー可能です。

NOTE

色付きテーマは装飾用です。実スキャンの確実性を優先する場合は Mono / Invert を推奨します。EC（エラー訂正レベル）はH(30%)が最も頑強ですが、データ容量は減ります。
